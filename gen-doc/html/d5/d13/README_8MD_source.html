1 # Reliable UDP
2 
3 A final project for the Gvahim program. This project implements a reliable stream above UDP,
4 called RUDP or Reliable UDP. The goal of this protocol is to allow a flexible and easily
5 adapted replacement in unreliable mediums in which TCP is inefficient.
6 The program architecture is based on proxying with RUDP, between end users, who communicate in TCP.
7 Each instance of the program is an asynchronous RUDP server, that includes:
8 a. A UDP socket for RUDP communication with other RUDP servers
9 b. TCP sockets for data communication with the end users.
10 c. TCP sockets for communication with the end users via control channel,
11  which allows request of statistics and requests to initialize connections.
12 d. TCP sockets for HTTP serving of a web page in which the user can
13 see statistics and request to initialize connections.
14 
15 The program works in the following way:
16 1. User A connects to RUDP Server A via control channel/web page, and asks to initialize
17 a connection, specifying the destination he wishes to communicate with (User B) and the additional RUDP Server
18 he wished to pass through (RUDP Server B).
19 2. Server A sends User A a port number.
20 3. User A connects to this port and the connection is initialized.
21 4. Packets User A sends will be sent to Server A in TCP, then to Server B in RUDP, and finally
22 to User B in TCP.
23 5. And vice versa.

26 ## Getting Started
27 
28 These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.
29 
30 ### Prerequisites
31 
32 In order to run the project there are some requirments:
33 ```
34 1) Download and install Cygwin: https://cygwin.com/install.html
35 2) In configurations of Cygwin install choose to install git and python 2.7
36 3) Enter the tmp file of cygwin: cd /tmp
37 4) Clone repository into tmp file: git clone https://github.com/EitanGronich/Reliable-UDP
38 ```
39 
40 ### Execution
41 
42 To execute the project open Command Prompt (or Terminal on Linux):
43 Reach the parent folder of this project:
44 ```
45 cd [location of Reliable-UDP]
46 ```
47 Running Server 1:
48 ```
49 python -m Reliable-UDP.Server (default: --rudp-port 1026 --control-port 1025 --http-port 80)
50 ```
51 
52 Running Server 2 (on different ports because both are on localhost):
53 ```
54 python -m Reliable-UDP.Server --rudp-port X --control-port Y --http-port Z
55 ```
56 
57 Now you can use the servers as drop-in replacement for TCP as described above.
